Breadcrumbs

Breadcrumbs indicate the current page's location within a navigational hierarchy.

A breadcrumb trail is composed of a <nav> element with an ordered list(<ol>) inside.

Each breadcrumb is a list item(<li>) that contains a link(<a>).

The current page is indicated with the aria-current="page" attribute.

This example assumes the user is at components.html#breadcrumbs.

Toast

Floating notifications that appear at the corner of the screen.

Toast notifications are ephemeral messages that disappear after a short duration.

The text content of the toast is logged to the console for debugging purposes.

Toast notifications are considered a subset of alerts and can be styled using .toast + { .success | .warning | .danger }.
