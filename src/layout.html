Aspect Ratio

The .aspect-ratio class is designed for media elements like images or videos. It maintains a specific aspect ratio and ensures content fits correctly.

The style parameter --aspect-ratio sets the aspect ratio. The default is 1 / 1.

The style parameter --aspect-ratio-padding sets the padding. The default is 0.

The style parameter --aspect-ratio-border sets the border. The default is 1px solid #000.

The style parameter --aspect-ratio-bg sets the background color. The default is transparent.

Card

The .card class adds padding and a border around content.

The style parameter --card-padding sets the padding inside the card. The default is 1rem.

The style parameter --card-border sets the border around the card. The default is 1px solid var(--text-color).

Carousel

The .carousel class creates a horizontal scrolling container with CSS scroll snapping.

The style parameter --carousel-gap sets the spacing between elements. The default is 1rem.

The style parameter --carousel-item-width sets the maximum width of each item in the carousel. The default is 80%.

Columns

The .columns class allows elements to switch between a horizontal and vertical layout based on a container width threshold.

While a standard responsive layout will wrap elements as the viewport shrinks, columns will switch entirely from horizontal to vertical without wrapping.

The style parameter --columns-gap sets the spacing between elements. The default is 1rem.

The style parameter --columns-min-width sets the threshold at which the elements will switch to a vertical stack. The default is 500px.

Container

The .container class centers content horizontally and sets a maximum width.

The style parameter --container-max-width sets the maximum width of the container. The default is 1200px.

The style parameter --container-padding sets the horizontal padding. The default is 1rem.

Grid

The .grid class creates a responsive grid of elements that wrap based on a minimum width.

The style parameter --grid-gap sets the spacing between elements. The default is 1rem.

The style parameter --grid-min-width sets the minimum width of each grid item. The default is 200px.

The style parameter --grid-align-items sets the vertical alignment of grid items. The default is end.

Overlay

The .overlay class positions an element absolutely within its closest positioned ancestor.

The ancestor must have position: relative and a defined height.

The style parameter --overlay-top sets the vertical position. The default is 50%.

The style parameter --overlay-left sets the horizontal position. The default is 50%.

The element is centered relative to these coordinates using transform: translate(-50%, -50%).

Sidebar

The .sidebar class creates a two-column layout with a main content area and a sidebar.

The main content area must have the class .main.

The sidebar content area must have the class .side.

The style parameter --sidebar-side-width sets the width of the sidebar. The default is 20rem.

The style parameter --sidebar-main-min-width sets the minimum percentage width of the main content before it wraps. The default is 50%.

The sidebar will automatically wrap below the main content when the available space is less than the combined preferred widths, making it container-agnostic.

The style parameter --sidebar-gap sets the spacing between the main content and sidebar. The default is 1rem.

Stack

The .stack class adds spacing between elements. Stacks are responsive and will wrap as the viewport shrinks.

The style parameter --stack-space sets the spacing between elements. The default is 1.5rem.

The style parameter --stack-padding sets the padding around the stack. The default is 0rem.

The class .horizontal can be added to create a horizontal stack.

The style parameter --stack-wrap can be used to control wrapping in horizontal stacks. The default is wrap.

The class .split can be added to push the last element to the end of the stack (bottom or right depending on orientation).
